# Freetube-redirect
Chromium extension to open YouTube links into FreeTube

## 🔧 How to Install

* Download the ZIP file above and extract it into a folder.
* Open Chrome and go to `chrome://extensions/`
* Turn on **"Developer mode"** (the toggle in the top-right corner).
* Click **"Load unpacked"**.
* Select the extracted `freetube-redirect` folder.
